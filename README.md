
# Module 18 - NoSQL Challenge: Social Network API

![Static Badge](https://img.shields.io/badge/license-MIT-green)

## Description

An attempt to develop an API for a social network web application where users can share their thoughts, react to friends' thoughts, and create a friend list.

## Installation

To install this application, perform the following from a terminal prompt:

 - Clone the repository
 - Change to the newly cloned repository directory
 - Run the following command
 ```
 npm install
 ```

## Usage

To use this application, perform the following from a terminal prompt:
- Change to the application directory
- Run the following command
```
npm run start
```
